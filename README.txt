(1) INTRODUCTION
-----------------
Follow Your Event is an App build in Apache and use java v 1.8.0_45.

Once you have installed and run this program, you have to go to http://localhost:8080/followyourevent

If you will use this with other server, and its server has a static IP Address, you have to consider that 
in some new web explorers the geolocation doesn't work, because security issues (https://sites.google.com/a/chromium.org/dev/Home/chromium-security/deprecating-powerful-features-on-insecure-origins).
To solve this you can use IE (Internet Explorer) best browser ever.
